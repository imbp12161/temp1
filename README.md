# 시수 캘린더 — 학원 등원·정산 관리 앱 (안드로이드)

학생별 **등원 요일 + 수업 시수(8/12/16/20회)** 로 등원일과 **정산일(수업 종료일)** 을 자동 계산하는 안드로이드 앱입니다.
공휴일은 자동 제외되고, **달력에서 날짜를 길게 눌러 끌면 방학을 한 번에 칠할 수 있습니다.**

---

## ⚠️ 먼저 읽어주세요 (APK 만드는 법)

이 프로젝트는 **소스 코드**입니다. 휴대폰에 설치하는 `.apk` 파일은 **컴파일(빌드)** 과정을 한 번 거쳐야 만들어집니다.
PC에 아무것도 설치하기 싫다면 **방법 1(권장)** 을, 직접 빌드하고 싶다면 **방법 2** 를 쓰세요.

---

## ✅ 방법 1 — GitHub에서 클라우드로 빌드 (설치 프로그램 0개, 권장)

GitHub가 클라우드에서 대신 빌드해 `.apk` 를 만들어 줍니다. 무료 GitHub 계정만 있으면 됩니다.

1. **GitHub 계정 만들기** : https://github.com 에서 가입(무료).
2. **새 저장소(Repository) 생성** : 오른쪽 위 `+` → `New repository` → 이름 입력(예: `sisu-calendar`) → `Create repository`.
3. **이 프로젝트 폴더 통째로 업로드**
   - 생성된 저장소 화면에서 `uploading an existing file` 링크 클릭.
   - 압축을 푼 이 폴더 안의 **모든 파일/폴더를 드래그&드롭** 으로 올립니다.
     (`app`, `gradle`, `.github`, `gradlew`, `build.gradle.kts`, `settings.gradle.kts` 등이 다 보여야 합니다.)
   - 아래 `Commit changes` 버튼 클릭.
4. **자동 빌드 시작** : 파일이 올라가면 위쪽 **`Actions`** 탭에서 빌드가 자동으로 돌아갑니다(보통 3~6분).
   - `Build APK` 워크플로우가 초록색 ✓ 가 되면 완료.
5. **APK 내려받기 (둘 중 편한 것)**
   - **(A) Releases 에서** : 저장소 우측 `Releases` → 최신 항목 → **`app-debug.apk`** 클릭해서 다운로드. ← 가장 쉬움
   - **(B) Actions 에서** : `Actions` → 방금 끝난 실행 클릭 → 맨 아래 `Artifacts` → `sisu-calendar-apk` 다운로드(zip) → 풀면 `app-debug.apk`.
6. 받은 `app-debug.apk` 를 휴대폰으로 옮겨 설치합니다. → 아래 **휴대폰 설치 방법** 참고.

> 빌드가 빨간 ✗ 로 실패하면, `Actions` → 실패한 단계 로그를 열어 **에러 메시지를 복사**해 주세요. 바로 고쳐 드립니다.

---

## 🛠 방법 2 — 내 PC의 Android Studio로 빌드

1. **Android Studio 설치** : https://developer.android.com/studio (무료).
2. Android Studio 실행 → `Open` → 이 폴더(`SisuCalendar`) 선택.
3. 처음 열면 Gradle 동기화 + 필요한 SDK 자동 다운로드가 진행됩니다(인터넷 필요, 몇 분).
4. 상단 메뉴 **`Build` → `Build App Bundle(s) / APK(s)` → `Build APK(s)`**.
5. 완료 알림의 `locate` 클릭 → `app/build/outputs/apk/debug/app-debug.apk` 가 결과물입니다.
6. 또는 휴대폰을 USB로 연결(개발자 옵션·USB 디버깅 켜기)하고 ▶(Run) 버튼으로 바로 설치 실행.

---

## 📱 휴대폰에 APK 설치하는 법

1. `app-debug.apk` 파일을 휴대폰으로 전송(카톡 나에게 보내기, USB, 구글 드라이브 등).
2. 파일을 탭하면 "출처를 알 수 없는 앱" 경고가 뜹니다 → **설정에서 해당 앱(파일 관리자/브라우저)에 "이 출처 허용"** 을 켠 뒤 다시 설치.
3. 설치 완료 후 홈 화면의 **시수 캘린더** 아이콘으로 실행.

> 디버그 APK라 설치 시 경고가 나오는 것은 정상입니다(개인 사용에 문제 없음). 정식 배포(플레이스토어)용으로 서명된 APK가 필요하면 알려주세요.

## 🎨 로고 바꾸기 (위치 지정 완료)

헤더 왼쪽 위에 표시되는 로고 파일 위치는 다음과 같습니다:

```
app/src/main/res/drawable/logo.xml   ← 현재는 임시 로고(자리표시자)
```

**내 로고 이미지로 교체하는 법**
1. `app/src/main/res/drawable/` 폴더의 **`logo.xml` 파일을 삭제**합니다.
2. 같은 폴더에 본인 로고를 **`logo.png`** 라는 이름으로 넣습니다.
   - 권장: **배경이 투명한 PNG**, 높이 **120px 이상**(가로로 긴 로고 권장).
   - 파일 이름은 반드시 소문자/숫자/밑줄(`_`)만, 이름은 `logo` 로 유지하세요. (예: `logo.png`)
3. 다시 빌드하면 로고가 바뀝니다.

> 임시 로고를 그대로 써도 됩니다. 앱 아이콘(런처 아이콘)을 바꾸려면 `res/drawable/ic_launcher_foreground.xml` 와 `ic_launcher_background.xml` 를 교체하면 됩니다.

## 🧭 앱 사용법 요약

- **달력은 한 달씩** 보이며, 상단의 `‹ 2026년 6월 ›` 좌우 화살표로 달을 넘깁니다. 가운데 연·월을 누르면 오늘 달로 돌아옵니다.
- **학생 등록** : 헤더의 `+` → 이름, 등원 요일(월~금), 수업 시수, 시작일, 색상 입력.
- **색상 의미** : 1차 사이클 파랑, 2차 초록(이후 교대). 사이클의 마지막 등원일(=**정산일**)은 자홍색 테두리로 강조됩니다.
- **공휴일** : 자동으로 등원에서 제외(빨간색). 학원이 운영하는 공휴일은 메뉴 → `공휴일 관리`에서 끌 수 있습니다.
- **방학 지정** : 달력의 날짜를 **길게 누른 뒤 손가락을 끌면** 그 범위가 방학으로 칠해집니다(다시 끌면 해제). 날짜를 짧게 탭하면 상세/단일 지정도 가능합니다.
- **상단 학생 칩** : `전체` 는 모든 학생을 점으로, 학생 이름을 누르면 그 학생만 자세히 봅니다.
- **백업/복원** : 메뉴 → 데이터 내보내기/불러오기 (JSON 파일). 데이터는 휴대폰에만 저장되니 기기 변경 전 백업을 권장합니다.
